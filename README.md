# creative NOVA – Next.js

[creative IDA](https://www.creative-ida.com/photography) 스타일을 참고한 포트폴리오 사이트입니다. Next.js 14 (App Router) + React로 구성되어 있습니다.

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000) 으로 접속하세요.

## 페이지

- **/** – Home (히어로 타이포그래피)
- **/works** – Works
- **/pictorial** – Pictorial
- **/photography** – Photography (갤러리 그리드)
- **/campaign** – Campaign
- **/about** – About NOVA

## 기술 스택

- Next.js 14 (App Router)
- React 18
- TypeScript
- CSS (미니멀 creative NOVA 스타일, Cormorant Garamond / Inter)
